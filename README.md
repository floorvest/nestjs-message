## NEST JS Message

Example to send message to users using nestjs

## Installation

import the example.sql first and copy .env.example to .env

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

```
